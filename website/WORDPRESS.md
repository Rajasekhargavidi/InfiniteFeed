# Add AI LinkedIn Marketing Agent Page to WordPress

## Method 1: Custom HTML Block (Recommended)

1. **Log in** to WordPress admin (wp-admin)
2. Go to **Pages → Add New**
3. Enter page title: **AI LinkedIn Marketing Agent**
4. Click the **+** button to add a block
5. Search for **"Custom HTML"** and select it
6. **Paste** the entire content from `wordpress-page-content.html`
7. Click **Publish** (or **Update** if editing)
8. Set the URL slug: e.g. `digital-marketing-agent` or `ai-linkedin-agent`

**Result:** `https://laravisionx.com/digital-marketing-agent/`

**Mobile app link to add:** https://rajasekhargavidi.github.io/InfiniteFeed/

---

## Method 2: Classic Editor

1. **Pages → Add New**
2. If you see blocks, switch to **Classic Editor** (install "Classic Editor" plugin if needed)
3. Enter title: **AI LinkedIn Marketing Agent**
4. Click the **Text** tab (not Visual)
5. Paste the content from `wordpress-page-content.html`
6. **Publish**

---

## Method 3: Full-Width Template

If your theme has a "Full Width" or "Blank" page template:

1. Create the page with Method 1 or 2
2. In the right sidebar, under **Page Attributes** or **Template**
3. Select **Full Width** or **Blank** (no sidebar)
4. Publish

---

## After Publishing

- Add the page to your **menu:** Appearance → Menus → add the new page
- Update **Contact** link in the page if your contact URL is different
- Test on mobile to ensure it looks good

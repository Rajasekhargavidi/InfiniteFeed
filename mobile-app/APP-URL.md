# Mobile App — Correct URL

## Use this URL (no /mobile-app/)

**https://rajasekhargavidi.github.io/InfiniteFeed/**

---

## Why /mobile-app/ gives 404

The GitHub Actions workflow deploys the `mobile-app` folder contents to the **root** of the site. The app is at the base URL, not in a subfolder.

| URL | Result |
|-----|--------|
| https://rajasekhargavidi.github.io/InfiniteFeed/ | ✅ App |
| https://rajasekhargavidi.github.io/InfiniteFeed/mobile-app/ | ❌ 404 |
